Proyecto 10 | Anio bisiesto

Explicaion

Este código solicita al usuario que ingrese un año y verifica si es un año bisiesto. Primero, valida la entrada para asegurarse de que sea un número positivo. Luego, usa una serie de condiciones para determinar si el año es bisiesto: un año es bisiesto si es divisible por 4, pero no es bisiesto si es divisible por 100, a menos que también sea divisible por 400. Finalmente, muestra un mensaje indicando si el año ingresado es bisiesto o no, según el resultado de las condiciones evaluadas.


            //declaro mis variables
var anio = parseFloat(prompt("Ingrese el anio"));

//utilizo if para dar un criterio y si es pues es, sino continua al otro
if( anio=anio%4 ===0 ){
    document.write("Es un anio bisiesto");
}else{
    document.write("No es un anio bisiesto");
}
