Proyecto 14 | Suma de Numeros Naturales

Explicaion

Este código solicita al usuario que ingrese un número y luego calcula la suma de todos los números naturales desde 1 hasta el número ingresado. Primero, se valida que el número ingresado sea mayor o igual a 1. Si es válido, se utiliza un bucle `for` que va sumando cada número natural al total acumulado en la variable `suma`. Al finalizar el bucle, se muestra el resultado de la suma de los números naturales hasta el número indicado por el usuario. Si el número ingresado no es válido, el código no realiza ninguna operación ni muestra ningún mensaje.


            //ingrese hasta que numero desea sumar los numeros naturales
            let num = parseFloat(prompt("Ingrese hasta qué número desea sumar los numeros naturales:"));
            
            // Verificar si el numero ingresado es mayor o igual a 1
            if (num >= 1) {
                let suma = 0;
                for (let i = 1; i <= num; i++) {
                    // Sumar cada número al total de la suma
                    suma += i;
                }  // Mostrar en pantalla la suma de los primeros 'num' numeros naturales
                document.write("La suma de los primeros ", num, " numeros naturales es: ", suma);
            }
