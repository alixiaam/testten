Proyecto 7 | Mayor o Menor

Explicaion

Este código solicita al usuario que ingrese su edad y luego determina si es mayor o menor de edad. Si el usuario ingresa una edad mayor o igual a 18, el programa muestra el mensaje "Usted es mayor". Si la edad ingresada es menor o igual a 18, muestra "Usted es menor". Si por alguna razón no se ingresa una edad válida, se muestra "edad no ingresada". Sin embargo, hay un pequeño error en la lógica, ya que el segundo `else if` debería ser solo `else`, porque abarca todos los casos restantes cuando la edad no es mayor de 18.


            flet edad = prompt("Ingrese su edad");

            if (edad >=18){
                document.write("Usted es mayor");
            }else if (edad <=18){
                    document.write("Usted es menor");
            }else{
                document.write("edad no ingresada");
            }
