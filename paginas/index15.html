Proyecto 15 | Conversion de Celsius y Fahrenheit

Explicaion

Este código solicita al usuario que ingrese si está trabajando con grados Celsius o Fahrenheit y luego pide el valor numérico de la temperatura que desea convertir. Utiliza un `if-else` para verificar si el usuario ingresó "Fahrenheit" o "Celsius" (admite varias formas de entrada como 'f', 'F', etc.). Si el usuario elige Fahrenheit, convierte la temperatura a Celsius utilizando la fórmula `(num-32)*(5/9)`. Si elige Celsius, convierte la temperatura a Fahrenheit utilizando la fórmula `(num*(9/5) + 32)`. Si la entrada no coincide con ninguna de las opciones válidas, se muestra un mensaje de error pidiendo al usuario que ingrese correctamente.


            //delcaro dos variables los dos son prompt pero uno es parseFloat porque es un numero

var grado = prompt("Estas ingresando Celsius o Fahrenheit?")
var num = parseFloat(prompt("Ingrese el monto de grado que deseas convertir"))
//aqui utilizo if para decir si el variable es igual a uno de esto criterios, haga esta formula
if(grado == 'f' || grado == 'F' || grado == 'fahrenheit' || grado == 'Fahrenheit'){
    var celsius=(num-32)*(5/9)
    document.write(num," grados Fahrenheit es igual a ",celsius," grado Celsius")
}else if(grado == 'c' || grado == 'C' || grado == 'celsius' || grado == 'Celsius'){
    var fahrenheit=num*(9/5) + 32
    document.write(num," grados celsius es igual a ",fahrenheit," grado fahrenheit")
}else{
    document.write("Error, porfavor escriba correctamente que se le pida")
}
