Proyecto 9 | Descuento en Compra

Explicaion

Este código solicita al usuario que ingrese el monto total de una compra y, utilizando una estructura switch, calcula el descuento aplicable según el valor ingresado. El monto total, almacenado en la variable Toral, es comparado con los rangos de precios definidos en cada caso del switch. Si el monto se encuentra dentro de uno de estos rangos, se calcula el descuento correspondiente: un 7% para montos entre 1000 y 1999, un 17% para montos entre 2000 y 2499, y un 20% para montos superiores a 2500. Si el monto no se ajusta a ningún rango definido, el descuento se establece en 0. Finalmente, el código muestra el monto del descuento aplicable o indica que no se aplicó descuento, según el caso encontrado.


            //Declaro mis variables
var Toral = parseFloat(prompt("Ingrese el monto total de la compra:"));
var descuento = 0;

// Determinar el descuento
switch (true) {
    case (Toral >= 1000 && Toral < 2000):
        descuento = 0.07 * Toral; 
        break;
    case (Toral >= 2000 && Toral < 2500):
        descuento = 0.17 * Toral; 
        break;
    case (Toral >= 2500):
        descuento = 0.20 * Toral; 
        break;
    default:
        descuento = 0;
        break;
}

// Mostrar el descuento aplicable
if (descuento > 0) {
    document.write("Usted tiene un descuento de L.",descuento.toFixed(2)," a la compra.");
} else {
    document.write("No se aplicó ningún descuento a la compra.");
}
