Proyecto 5 | Factorial

Explicaion

Este código calcula el factorial de un número ingresado por el usuario. Primero, se declara la variable `num` que almacena el valor que el usuario ingresa a través de un cuadro de diálogo (`prompt`) y se convierte a número con `parseFloat()`. Luego, se inicializan las variables `fac` (factorial) y `val` en 1. Utilizando un bucle `do-while`, el código multiplica el valor de `fac` por `val` en cada iteración y luego incrementa `val` en 1, hasta que `val` sea mayor que `num`. Finalmente, se imprime el resultado del factorial utilizando `document.write()`.


            //declaro mis variables
    var num = parseFloat(prompt("Ingrese un valor que deseas encontrar en factorial de"))
    var fac = 1
    var val = 1

            //se utiliza do while para hacer algo si esta condicion es valida
    do{
        fac *= val;
        val++;
    }while(val<=num);

        //Se imprima el resultado
    document.write("El factorial de ",num," es igual a ",fac);
