Proyecto 13 | Tabla de Multiplicacion

Explicaion

Este código solicita al usuario que ingrese un número para generar su tabla de multiplicar del 1 al 10. Primero, convierte la entrada del usuario en un número entero utilizando `parseInt`. Luego, verifica si el número es válido (es decir, no es `NaN` o 0). Si el número es válido, el código usa un bucle `for` para iterar del 1 al 10, calculando y mostrando el resultado de la multiplicación del número ingresado por cada valor del bucle. Si el usuario no ingresa un número válido, se muestra un mensaje indicando que se debe ingresar un número válido.


            // Solicitar al usuario que ingrese un numero
let numero = parseInt(prompt("Ingrese un numero para generar su tabla de multiplicar del 1 al 10:"));

if (numero) {
    //declarando 1 al 10 para generar la tabla de multiplicar
    for (let i = 1; i <= 10; i++) {
        //imprimacion de tabla
        document.write(" ",numero + " x " + i + " = " + numero * i);
    }
} else {
    document.write("Debe ingresar un numero valido.");
}
