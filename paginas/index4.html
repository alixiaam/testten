Proyecto 4 | Primo o No

Explicaion

Este código verifica si un número ingresado por el usuario es primo. Primero, solicita al usuario que ingrese un número y lo convierte a un valor numérico. Luego, comprueba si el número es mayor que 1 (porque los números primos son mayores que 1). Si lo es, asume inicialmente que el número es primo y utiliza un bucle para verificar si es divisible por algún número entre 2 y la raíz cuadrada del número ingresado. Si encuentra un divisor, cambia la variable `esPrimo` a `false` y sale del bucle. Después del bucle, si `esPrimo` sigue siendo verdadero, muestra que el número es primo; si es falso, indica que no lo es. Si el número ingresado no es mayor que 1, muestra un error.


               //declarar variables, el numero que el usuario ingrese
   let numero = parseFloat(prompt("Ingrese un numero para verificar si es primo o no:"));
   
   if (numero > 1) {
       let esPrimo = true;
        // Recorrer los numeros desde 2 hasta la raiz cuadrada del numero ingresado
       for (let i = 2; i <= Math.sqrt(numero); i++) {
           // Verificar si el numero es divisible por i y si es divisible, cambiar esPrimo a falso y salir del bucle
           if (numero % i === 0) {
               esPrimo = false;
               break;
           }
       }
        // Verificar si esPrimo sigue siendo verdadero
       if (esPrimo) {
           // Si es verdadero, escribir que el numero es primo o vice versa
           document.write(numero, " es un numero primo.");
       } else {
           document.write(numero, " no es un numero primo.");
       }
   } else {
       document.write("Error");
   }
